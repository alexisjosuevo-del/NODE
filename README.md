# NODE_web (sitio estático)

## Cómo ejecutar local
1) Abre `index.html` en tu navegador
o
2) Levanta un servidor:
   - Windows: `python -m http.server 8000`
   - Luego abre: http://localhost:8000

## Cómo subir a GitHub Pages
- Sube TODO el contenido de esta carpeta a tu repo (raíz)
- Settings → Pages → Deploy from branch → main / root

## Formulario de contacto
Este demo guarda el lead en `localStorage`.
Si quieres recibirlo por correo:
- Opción A: Formspree (front-only)
- Opción B: Netlify Forms
- Opción C: tu backend (Firebase Functions / Node API)

## Personalización rápida
- Textos: `index.html`
- Estilos: `css/styles.css`
- Interacciones (tabs/slider): `js/main.js`
- Logo/Favicon: `assets/logo.jpg` y `assets/favicon.png`, `favicon.ico`


## Video de portada
- Reemplaza `assets/hero.mp4` por tu video (mismo nombre) para cambiar el fondo del hero.
